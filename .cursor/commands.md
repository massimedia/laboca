# LABOCA AI Commands

These commands help generate common ecommerce features.

---

## /create-service

Create a Medusa service inside the backend.

Requirements:
- Follow Medusa service architecture
- Use dependency injection
- Keep business logic inside services
- Add TypeScript types

---

## /create-workflow

Create a Medusa workflow for complex backend operations.

Requirements:
- Use Medusa workflows
- Keep workflows modular
- Include validation and error handling

---

## /create-api-route

Create a backend API route.

Requirements:
- Follow Medusa routing patterns
- Delegate logic to services
- Keep routes thin

---

## /create-react-component

Create a reusable React component for the storefront.

Requirements:
- Functional React component
- Use hooks
- Keep UI modular
- Follow ecommerce UI patterns

---

## /create-product-page

Create a storefront product page.

Requirements:
- Fetch product data from Medusa
- Display product info
- Support variant selection
- Support add-to-cart

---

## /debug-medusa

Analyze backend code and identify Medusa architecture issues.
